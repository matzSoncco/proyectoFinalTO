
Simulador interactivo que representa digitalmente el plano de un edificio, donde agentes (personas y rescatistas) se mueven hacia salidas de emergencia según distintos parámetros, aplicando conceptos avanzados de Programación Orientada a Objetos.


### Compilar y Ejecutar

```bash
# Opción 1: Con CMake (recomendado)
mkdir build
cd build
cmake ..
make
./simulador_agentes


## Pruebas

Ejecutar `./simulador_agentes` para ver la demostración que incluye:
1. Creación de diferentes tipos de agentes
2. Clonación con Patrón Prototype
3. Notificación de eventos con Observer
4. Simulación de evacuación
5. Reporte de estadísticas
